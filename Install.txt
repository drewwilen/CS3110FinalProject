In order to run you will need to install a few packages to do so you can run the following commands in your terminal:

opam install ANSITerminal


To access scraping capabilities you must run the following commands: 

opam install tls-lwt
opam install Yojson
opam install cohttp
opam install cohttp-lwt-unix  
opam install csv
brew install cloc (if you want to run the make cloc command for lines of code information)


 
If you would like to run the program expand your terminal and call 'make platform'
