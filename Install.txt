In order to run you will need to install a few packages to do so you can run the following commands in your terminal:

opam install ANSITerminal

After you have installed ANSITerminal you can run the program in the terminal by calling make run

To run scraping you must run the following commands: 

opam install Unit
opam install tls-lwt
opam install Yojson
opam install cohttp
opam install cohttp-lwt-unix  
 
If you would like to run it with th emake file you should edit the make run config to be OCAMLRUNPARAM=b dune exec Scraping/main.exe
